# 🏥 SedAirCheck - ワンクリックデプロイ

消化器内科の先生向けの簡単デプロイガイドです。

## 🚀 最も簡単な方法

### ⚡ ワンクリックデプロイ

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/[YOUR_USERNAME]/SedAirCheck&env=OPENAI_API_KEY&envDescription=OpenAI%20API%20Key%20for%20image%20analysis&envLink=https://platform.openai.com/api-keys)

**手順:**
1. 上のボタンをクリック
2. GitHubでログイン
3. OpenAI APIキーを入力
4. "Deploy" をクリック
5. 完了！URLが発行されます

## 📱 アクセス方法

デプロイ完了後、生成されたURL（例: `https://sed-air-check-xxx.vercel.app`）にアクセスしてください。

### スマートフォンから
- SafariやChromeで上記URLを開く
- ホーム画面に追加すればアプリのように使用可能

### PCから
- 任意のウェブブラウザで上記URLを開く

## 🔑 OpenAI APIキーの取得方法

1. [OpenAI Platform](https://platform.openai.com) にアクセス
2. アカウント作成・ログイン
3. API Keys ページでキーを生成
4. 生成されたキーをVercelの環境変数に設定

## 💡 使用例

1. **画像準備**: iPhoneで口腔内写真を撮影
2. **アップロード**: ウェブサイトで画像を選択
3. **結果確認**: 約3-5秒でマランパティ分類が表示
4. **記録**: 結果をスクリーンショットまたはメモで保存
